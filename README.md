# Europe Trip 2026 - Interactive Map

## Quick Deploy to Vercel (Easiest - 2 minutes)

1. Go to [vercel.com](https://vercel.com) and sign up (it's free)
2. Click "Add New Project"
3. Choose "Import Git Repository" or "Upload files"
4. If uploading: zip this entire folder and upload
5. Vercel will auto-detect it's a Vite project
6. Click "Deploy"
7. Done! You'll get a URL like `your-project.vercel.app`

## Alternative: Deploy to Netlify

1. Go to [netlify.com](https://netlify.com) and sign up (free)
2. Drag and drop this folder onto their deploy page
3. Done!

## Run Locally First (Optional)

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## Build for Production

```bash
npm run build
```

The `dist` folder will contain your production-ready files.

## What's Included

- Interactive timeline navigation
- Three map modes (Atlas, Road, Terrain)
- Route visualization with flight, train, and driving segments
- Location details cards
- Fully responsive design

Enjoy your trip planning!
